## how to work with this repo:

1. `npm init -y`
2. `npm i express dotenv`
3. `npm i -D supertest jest`

**On the `package.json` change the `script` to**

```
"scripts": {
    "start": "node index.js",
    "test": "jest --verbose --coverage"
}

```
